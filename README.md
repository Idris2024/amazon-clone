# Amazon Clone (JavaScript)

A simple Amazon-like e-commerce project built with vanilla JavaScript.

## Features
- Home page
- Add to cart
- Checkout
- Place orders
- Orders & tracking page
- LocalStorage support

## How to run
1. Clone the repo
2. Open `amazon.html` in a browser

## Tech stack
- HTML
- CSS
- JavaScript
